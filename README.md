# ESSENCE · Paris — промо-страница

Статический лонгрид (один файл `index.html`). Никакого сервера не нужно —
кладётся на GitHub Pages или любой статический хостинг.

## Структура папок

```
SITE/
├── index.html          ← страница (готова)
├── img/
│   ├── hero/           ← кадры для hero, формат 16:9 (широкие)
│   ├── collection/     ← 6 флаконов, формат 4:5 (вертикальные)
│   ├── format/         ← диптих 100 ml / 50 ml, формат 3:4
│   └── maison/         ← интерьеры-храм, форматы 4:5 и 16:9
├── video/              ← mp4 ролика (для секции Le Film)
├── fonts/              ← опционально: локальные шрифты
└── README.md           ← этот файл
```

## Как вставить своё изображение

Найди в `index.html` нужный блок-плейсхолдер — это `<div class="ph ...">`.
Замени его содержимое так:

**Было (плейсхолдер):**
```html
<div class="ph r4x5">
  <div class="tag">...подпись...</div>
</div>
```

**Стало (с картинкой):**
```html
<div class="ph r4x5 filled" style="background-image:url(img/collection/n92.jpg)"></div>
```

То есть: добавить класс `filled`, добавить `style="background-image:url(путь)"`,
и удалить строку `<div class="tag">…</div>`. Подпись исчезнет, встанет картинка.

## Видео (секция Le Film)

Найди блок с классом `dark-ph` в секции `#film` и замени на:
```html
<video autoplay muted loop playsinline poster="img/hero/poster.jpg"
       style="width:100%;aspect-ratio:16/9;object-fit:cover">
  <source src="video/essence-film.mp4" type="video/mp4">
</video>
```

## Оптимизация картинок (важно для скорости)

Лонгрид с тяжёлыми PNG будет грузиться долго. Перед заливкой сожми в JPG/WebP:
- ширина ~1600–2000 px достаточно
- вес каждого кадра — до 300–500 КБ
- формат WebP предпочтителен (меньше вес при том же качестве)

## Заливка на GitHub Pages (кратко)

1. Репозиторий: `pfdesign3d/essence-paris`, Public.
2. `main`, содержимое папки SITE в корне (index.html + img/ + video/ + CNAME).
3. Settings → Pages → Source: `main`, папка `/root` → Save.
4. Через 1–2 минуты сайт живёт на `https://pfdesign3d.github.io/essence-paris/`.

### Свой домен — essence-paris.store
5. Домен куплен: `essence-paris.store`.
6. Файл `CNAME` в корне репозитория уже содержит `essence-paris.store`.
7. У регистратора домена прописать A-записи на голый домен (essence-paris.store):
   `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
   (www можно оставить как редирект на голый домен через настройки регистратора).
8. Settings → Pages → Custom domain → `essence-paris.store` → Save,
   дождаться зелёной галочки DNS check → включить Enforce HTTPS.
   SSL-сертификат GitHub выпускает и продлевает сам — вручную ничего
   устанавливать не нужно.

## Шрифты

Сейчас страница тянет шрифты из Google Fonts (Cormorant, Jost, Space Mono) —
интернет нужен только при загрузке страницы у посетителя. Если хочешь полную
автономность, скачай эти шрифты в `fonts/` и замени `<link>` на `@font-face` —
скажи, помогу переписать.
